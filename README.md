## 使用技術
- 言語: TypeScript 3.9.6
- フロントエンド: React 16.13.1, Redux, React Router, Material UI
- バックエンド: Express 4.17.1, Sequelize
- インフラ: Heroku
- 他: axios

## 機能一覧
- 家事保存・削除
- 作成日降順一覧
- 作業日降順一覧
- 分類別集計一覧
